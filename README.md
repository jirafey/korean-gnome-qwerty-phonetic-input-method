### Composition Rules
1. **Jump Logic:** When a syllable ends in a consonant (e.g., `z-a-n`), the engine holds it as a batchim.
2. If the next key is a vowel, that consonant automatically moves to the next block.
3. **Double Consonants:** Double consonants are triggered by tapping the same consonant key twice (e.g., `ss` for ㅆ).
4. **Double Vowels:** Iotated vowels (the "Y" sounds) can be reached via dedicated keys (f, y, v, q) OR by double-tapping their base vowel (aa, ll, oo, uu).
